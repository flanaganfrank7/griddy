Hagrid

Hagrid is a simple grid script built for designers and developers on top of P5.JS, and inspired by Basehold.it and hosted with Netlify.

Usage

Embed the following scripts into the bottom of your html file, after the body.

    
    <script src='https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.10.2/p5.min.js'></script>
    <script src="https://hagridjs.netlify.com/functions/function.js" color="#f0f0f0" size=8 zed=-10></script>
Attributes

Hagrid takes three parameters:

  1. color: set the color of the grid by color name, hex value, rbg value, etc.
  2. size: set the size of the grid cell
  3. zed: set the z-index value of the grid
